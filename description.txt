Цели
    1. Проходя курс по HTML / CSS накидать свою интерактивную шпаргалку по JavaScript.
    2. При получении новых знаний улучшать сайт, доводить его до нужного состояниия.
    3. Интерактивность реализовать в возможности повторять материал по минимальному количеству раз. При достижении этого количества либо исключить данную тему из списка обучения, либо запустить на повторный цикл. Сделать по аналогии с Lingua Leo.

Предварительная планировка JavaScript
    Система обучения / напоминания
